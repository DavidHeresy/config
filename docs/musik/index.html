Musik

Musik ist ein auditorisches Muster.
Auditorisch bedeutet wir können es hören.
Das Muster entsteht durch einzelne Töne.

Ein Ton ist eine Funktion L(t,f).
L ist die Lautstärke zur Zeit t an der Frequenz f.
Der Kammerton A4 hat die Frequenz 440Hz.

Oktaven

Der Frequenzraum ist in Oktaven unterteilt.
Mit jeder Oktave verdoppelt sich die Frequenz.
Die 4. Oktave geht von 261Hz bis 523Hz.

Eine Oktave ist in 12 Halbtonschritte unterteilt.
Mit jedem Halbtonschritt erhöht sich die Frequenz um die zwölfte Wurzel von 2.
Ein Ganztonschritt ist eine Erhöhung um zwei Halbtonschritte.

Noten

Noten sind die Namen der Halbtonschritte.
Sie entsprechen den Tasten auf einem Klavier.
Sie heißen C C#/Db D D#/Eb E F F#/Gb G G#/Ab A A#/Bb B.

Ein SoundFont ist ein auditorischer Schriftsatz für Noten.
Zu jeder Note gibt es einen Ton.
Der Ton wird von der Frequenz der Note dominiert.

Akkorde

Ein n-Ton ist die Kombination von n Tönen.
Ein 2-Ton ist ein Interval.
Ein 3-Ton ist ein Akkord.

Melodien

Eine Melodie ist eine Folge von Tönen.
Eine Folge des gleichen Tons ist ein Rythmus.
Eine n-Tonleiter ist eine Auswahl von n Tönen aus einer Oktave.
