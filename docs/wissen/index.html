Wissen

Auf dieser Seite sammle ich mein Wissen.

Was ist Wissen?

Wissen ist Information, die für mich wertvoll ist. Das Ziel dabei ist, Wissen zu verstehen und zu speichern.

Was ist Information?

Informationen sind kodierte Daten.
Dieser Text ist in Deutsch kodiert.

Was sind Daten?

Alles, was messbar ist, enthält Daten.

Was ist eine Messung?

Eine Messung überträgt Daten aus einem beliebigen System in einen Speicher.

Was ist eine Kodierung?

Kodierung ist der allgemeine Begriff für Sprache. Eine Kodierung ohne Interpretations-Spielraum kann als deterministisch bezeichnet werden. Ein Kodierung mit Interpretations-Spielraum kann als frei bezeichnet werden. In einer freien Kodierung kommt es zu Fehlern, in einer deterministischen Kodierung gibt es keine Fehler.

Was bedeutet "wertvoll"?

Wertvoll ist ein subjektives Adjektiv. Es beschreibt, wie sehr ich bereit bin, für eine Sache Arbeit zu verrichten.

Was ist Verständnis?

Verständnis ist eine subjektive Fähigkeit. Sie beschreibt, wie sehr ich in der Lage bin, Wissen auf ein Problem anzuwenden.
